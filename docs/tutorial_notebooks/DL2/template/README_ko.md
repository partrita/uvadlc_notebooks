# 딥 러닝 2 과정의 노트북 튜토리얼 템플릿

이 폴더는 암스테르담 대학교의 딥 러닝 2 과정을 위한 새로운 노트북 튜토리얼을 만드는 템플릿을 나타냅니다. 튜토리얼 작성 방법에 대한 지침은 `TemplateNotebook.ipynb`를 참조하십시오. 아래에서는 처음부터 새로운 튜토리얼을 만들고 나중에 이 저장소와 웹사이트에 추가하는 방법에 대한 지침을 제공합니다.
* 이 저장소를 포크합니다.
* 포크에서 이 `template` 폴더를 복제하고 새 폴더의 이름을 튜토리얼 폴더로 바꿉니다 (노트북당 하나의 폴더를 원합니다).
* 새 폴더에서 이 README와 `example_image.svg` 파일을 제거합니다. 노트북에 이미지를 추가하려면 (권장) 동일한 폴더에 추가하십시오. 템플릿은 이미지를 포함하는 방법의 예를 제공합니다.
* 튜토리얼을 마치면 노트북을 위에서 아래로 완전히 새로 실행하십시오. 모든 셀의 출력이 웹사이트에 표시되므로 모든 것이 원하는 대로 되었는지 확인하십시오.
* 가장 바깥쪽 폴더로 이동한 다음 `docs/index.rst` 파일로 이동합니다. 파일 끝에 모든 노트북 목록이 표시됩니다. '딥 러닝 2' 탭 아래에 노트북 경로를 추가합니다. 이렇게 하면 웹사이트에 추가됩니다 (그렇지 않으면 무시됩니다).
* 웹사이트를 로컬에서 빌드하는 것이 좋습니다. `docs/requirements.txt` 파일을 통해 필요한 패키지를 설치할 수 있습니다. `docs/`로 이동하여 `make html`을 실행합니다. 그런 다음 `_build/html/` 폴더에서 간단한 파이썬 서버 `python -m http.server`를 시작하고 웹사이트를 볼 수 있습니다. 모든 것이 원하는 대로 표시되는지 확인하십시오.
* 마지막으로 포크된 저장소에서 원래 uvadlc_notebooks 저장소의 `master` 브랜치로 풀 리퀘스트를 만듭니다. 저장소에 추가하기 전에 노트북 변경 사항을 논의할 수 있습니다.
